
# Полностью рабочий “ручной” загрузчик
(bootimage повторяет эту сборку автоматом)

Что реально меняется при переходе 32 → 64 бит

Multiboot2 не загружает 64-битное ядро напрямую.

GRUB делает:

`BIOS → 32-bit protected mode → ваш _start (32-bit!)`

Вы обязаны:
* стартовать в 32-bit
* включить:
    * PAE
    * paging
    * 64-битный long mode
* сделать far jump в 64-битный сегмент
* и только потом войти в 64-bit Rust-код (запускаем Rust entry (kernel_main))

То есть архитектура становится двухэтапной.
